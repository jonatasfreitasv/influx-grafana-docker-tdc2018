# Node + Influx + Grafada

## Requisitos

- Node
- Docker

## Rodar Influx + Grafana

`docker compose up`

## Rodar aplicação

```
cd /app
npm install
npm start
```

## Acessar Grafana:

http://localhost:3000

## Configuração Influx:

```
Name: Influx
URL: http://node-influx-grafana_influxdb_1:8086
Database: tdcpoa2018
```
